# Coderpad

## Key bindings

- Ctrl+Enter : Runs code
- Ctrl+D     : Add next match to selection
- Ctrl+Space : Autocompletion

## Settings

- autocompletion parentheses/brackets (WATCH OUT! doesn't complete quotes/do-end)
