# Latex

## Basics

- math expressions: surrounded with `$`
- comments: start with `%`

## Math chars/expressions

```latex
x^2             % exponentiation/superscript
x_1 E_{r,c}     % subscript

\sqrt{x}        % square root
\vec{V}         % vector (overhead arrow)
\sum{x}         % summation

\langle \rangle % angle brackets
\alpha ...      % greek letters
\cdot           % multiplication dot
```
